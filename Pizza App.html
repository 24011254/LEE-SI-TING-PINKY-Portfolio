Problem Statement: Build a Pizza Menu Website for Your Pizza Shop


Objective


Create a simple, responsive web page using React to display a pizza menu for your pizza shop. The web page should display a list of available pizzas, each with a name, ingredients


The website should use various React concepts, including props, conditional rendering, array mapping,


Requirements


  1. Pizza Component with Props
    • Create a Pizza component that displays the pizza's name, description, price, and image. Use props to pass these details into the component so that it can be reused for multiple pizzas.

  2. Pizza Data in an Array
    • Store all pizza data (name, ingredients, price, and image URL) in an array. Use the map() function to loop through this array and render each pizza item on the web page dynamically using the Pizza component.

  3. Conditional Rendering for Shop Status
    • The shop is open only between 10 am and 10 pm. Use conditional rendering to display different content based on the current time:
      • If the shop is open:
        • Display "We're currently open" in the footer.
        • Show an "Order" button below the message.
        • Display the tagline "Authentic Italian Cuisine" at the top.
      • If the shop is closed:
        • Display "Sorry, we're closed" in the footer.
        • Do not display any tagline at the top.

  4. Styling and Semantic Markup
    • CStyle the page to resemble a simple, clean menu layout with a modern feel, like the sample.
    • Use appropriate semantic HTML tags to improve the website’s accessibility and structure.
      https://www.semrush.com/blog/semantic-html5-guide/

  5. Component Extraction
    • As the application grows, extract long sections of code into new components for better code organization and readability. For example, create an Order component to display the "We're currently open" message and the "Order" button, then render this component in the Footer when the shop is open.

  6. Customization and Extended Features
    • Customize the app by applying unique styles and layout designs to give the UI a distinctive look. Be creative and experiment with colours, typography, and layout.
    • Extend the functionality of the website by adding new features or interactive elements, such as a "Favourites" feature for pizzas, a simple "Search" filter, or any additional feature that enhances the user experience.

  7. Hosting the App Online
      Host your React application on a platform like GitHub Pages, Vercel, or Netlify so that it is accessible online. Share the live link to demonstrate your completed project.


Additional Notes

  • Focus only on the frontend part of the website. There is no need to implement any backend or database functionality.
  • Use CSS for styling and ensure that the layout is responsive across different devices.
  • Structure your project well and ensure code readability and reusability.

Expected Outcome


A visually appealing, unique pizza menu web page that showcases each pizza with its details and dynamically updates based on the time of day to show whether the shop is open or closed. Additionally, the page should demonstrate your creativity in styling, additional functionalities, and the ability to host a project online.
