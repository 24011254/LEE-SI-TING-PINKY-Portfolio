Problem Statement: Create a Travel Packing List Application


Objective


You are tasked with developing a React-based web application called "My Travel List." This app helps users manage their travel packing lists by adding items, marking them as packed, and removing them when necessary.


Requirements


  1. Core Features:
    • Add Items: Users can add items with a quantity and description via a form.
    • Display List: Show all items in a packing list, including their status (packed/unpacked).
    • Mark Packed: Allow users to mark items as packed by toggling a checkbox.
    • Delete Items: Enable users to remove items from the list.

  2. Technical Expectations:
    • Implement state management using the useState hook to track the list of items.
    • Use props to pass data and functions between components.
    • Employ JSX to design an intuitive and clean UI.
    • Include a footer that dynamically calculates and displays the percentage of packed items.

  3. Components and Functions:
    • App Component:
      • Manage the overall state of the application.
      • Define handlers for adding, updating, and deleting items.
    • Form Component:
      • Provide a form for users to input item details.
      • Use state to handle form inputs and reset them after submission.
    • PackingList Component:
      • Render a list of items using the Item component.
    • Item Component:
      • Display individual item details with a checkbox and delete button.
      • Handle updates and deletions via callback functions.
    • Stats Component:
      • Show statistics about the total items, packed items, and packing completion percentage.

  4. Application Structure:
    • Application Structure:
      • Use components to modularize the application: Logo, Form, PackingList, Item, and Stats.
      • Implement proper parent-child relationships to share state and props effectively.

  5. Customization and Extended Features
    • Customize the app by applying unique styles and layout designs to give the UI a distinctive look. Be creative and experiment with colours, typography, and layout.
    • Extend the functionality of the website by adding new features or interactive elements, such as sorting options, a simple "Search" filter, or any additional feature that enhances the user experience.

  6. Hosting the App Online
    • Host your React application on a platform like GitHub Pages, Vercel, or Netlify so that it is accessible online. Share the live link to demonstrate your completed project.

Outcome

By completing this task, you will demonstrate the understanding of:

  • Building and organizing components in React.
  • Using JSX for UI elements.
  • Passing data and functions via props.
  • Managing state effectively with React hooks.
  • Creating functional and interactive forms.
